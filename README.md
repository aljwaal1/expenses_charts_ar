# مصاريفي

تطبيق Flutter Android خفيف مبني على قالب APK المباشر.

## البناء

يتم البناء عبر GitHub Actions فقط من خلال:

`.github/workflows/build-apk.yml`

## ملف APK

بعد نجاح البناء ستجد الملف هنا:

`apk/personal-expenses.apk`

## ملاحظات

- لا يوجد AAB.
- لا يوجد Release artifact.
- النسخة الأولى خفيفة وقابلة للتطوير.
